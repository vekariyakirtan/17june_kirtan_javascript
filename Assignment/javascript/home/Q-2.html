What is the use of isNaN function?

NaN (NOT A NUMBER) :

JavaScript's isNaN() function checks whether a given value is an illegal number. If true, it returns the value, otherwise, it returns false.

the isNan() function is used to deteemines whether a value is an illegal number (Not a number).

the global isNan() function, converts the tested value to a number, then test it.

Example

syntax : isNan(value)

<script>
let a = 10;
let b = 30.44;
let c = "Hello"

if(isNan(a)){
document.write("not number"); true
}
else{ document.write("legal number"); false
}

(OUTPUT : LEGAL NUMBER)
</script>
