What is === operator?

A strict equality operator (===) decides whether two operands are equal and returns a Boolean result, unlike equality operators that consider operands of different types as different.

Example

<script>
let a = 50;
let b = 100

if(a === b){
document.write("true");
}
else{
document.write("flase");
}

(OUTPUT : TRUE)
</script>
