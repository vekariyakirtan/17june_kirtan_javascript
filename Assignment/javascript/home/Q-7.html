How can the style/class of an element be changed?

Using JavaScript, you can manipulate the DOM (Document Object Model) to change the style or class of an HTML element. It is possible to do this in a few different ways.

Hide and Show the Para

Example

<script>
const btn_hide = document.querySelector("#hide");
const btn_show = document.querySelector("#show");
const btn_color = document.querySelector("#color");

const para = document.querySelector("#p");

btn_hide.addEventListener('click' , () => {
para.classList.add("hide");
});

btn_show.addEventListener('click' , () => {
para.classList.remove("hide");
});

</script>
