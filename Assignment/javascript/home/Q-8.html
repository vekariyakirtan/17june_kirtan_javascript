What are all the looping structures in JavaScript?

Loop is use to execute the line of statement multiple time

1 : Entry Control Loop

while loop

for loop

for-in loop

for-of loop

for each loop

2 : Exit Control Loop

Do while loop is exit control loop. Exit control loop by default execute one time.

Do-while loop

Example

<script>

while(i <= 10){
console.log(i);
i++;
}
