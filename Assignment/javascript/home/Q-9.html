How can you convert the string of any base to an integer in JavaScript?

javascript's parseInt() function converts a string to an integer when it doesn't contain a number. If the string is accompanied by a number, then only that number is returned.

Example

<script>
function table(number){
for(i = 1 ; i<= 10 ; i++){
const res = i*number;
console.log(`${number} x ${i} = ${res}`);
}
}
table(33);

</script>
