What is jQuery?

jQuery is a JavaScript Library.

jQuery is a lightweight JavaScript library that lets you write less and do more. It simplifies the use of JavaScript on your website.

jQuery simplifies a lot of JavaScript tasks by wrapping them in methods you can call together instead of using several lines of code.
